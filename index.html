Markdown用法

标题排列

“#”表示一级标题,以此类推

图片与链接

编辑格式 中括号里边写图片或链接名字,后边紧跟小括号写图或链接的地址 图片的中括号前需加上!

图片为: 

 [本地测试图片](/images/testImage.jpeg)

本地图片需要点击才会显示,暂时未找到比较好的方法

链接为: markdown使用

插入图片的地址需要图床，这里推荐围脖图床修复计划 与 CloudApp 的服务，生成URL地址即可。

测试过 直接百度图片的链接就可以 但是不知道是不是偶然情况(23333….)

字体设置

一个*包含的文字为斜体

两个*包含的文字为粗体

代码框

需要用两个`把中间代码包裹起来

NSString *regex = @"^(?![0-9]+$)(?![a-zA-Z]+$)[0-9A-Za-z]{6,20}$";

<code>是行内代码(inline code)的写法

还有块代码(block code)的写法：代码每一行的前面都加4个空格或一个tab

还有一种Fenced Code Block，写法是：第一行和最后一行都是3个 “ ` “，中间的行是代码，
